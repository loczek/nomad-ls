## Overview

A simple [nomad](https://developer.hashicorp.com/nomad) language server

## Features

- autocompletion
- diagnostics
- hover information
- driver support (docker, exec, raw_exec, qemu, java)

### Building

```shell
$ go build -o ./bin/nomad-language-server ./main.go
```
